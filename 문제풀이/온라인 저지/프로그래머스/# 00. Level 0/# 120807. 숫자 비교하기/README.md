# Programmers 120807 : 숫자 비교하기
- 난이도 : Level 0
- 두 수가 같으면 1 다르면 -1을 retrun하도록 solution 함수를 완성하라.
- 문제 : [링크](https://school.programmers.co.kr/learn/courses/30/lessons/120807)

---

## 풀이
```java
public class Solution {
    public int solution(int num1, int num2) {
        return (num1 == num2)
                ? 1
                : -1;
    }
}
```
