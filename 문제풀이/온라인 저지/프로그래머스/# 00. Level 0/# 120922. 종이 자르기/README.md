- 난이도 : Level 0
- 정수 M, N이 매개변수로 주어질 때, M x N 크기의 종이를 최소로 가위질 해야 하는 횟수를 return

# 풀이
```java
public class Solution {
    public int solution(int M, int N) {
        return M * N -1;
    }
}
```
