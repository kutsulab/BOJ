- 제목 : 팀 나누기
- 난이도 : Bronze 4
- 두 팀의 스킬 레벨 차이의 최솟값을 출력

# 필요 지식
- 없음

---

# 풀이
```python
a, b, c, d = map(int, input().split())
print(abs((a + d) - (b + c)))
```
- 숫자들은 감소하지 않는 수열로 이루어져 있다.
- 첫값과 끝값/ 가운데 두 값끼리 묶어주면 비교적 균등한 값 분배가 될 것이다.
- 이 둘의 차는 음수일 수 있으므로, abs 를 통해 절댓값 처리한다.
