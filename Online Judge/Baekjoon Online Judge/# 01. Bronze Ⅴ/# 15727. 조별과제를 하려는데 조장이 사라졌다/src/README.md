## Main.java : 백준 제출 답안
- 입력은 BufferedReader
- 사용자 입력을 받은 값을 int로 파싱 후 `int L`에 저장
- L을 5로 나눈 나머지가
  - 0이면 : t = L/5
  - 0이 아니면 : t = L/5 + 1
- t 출력
